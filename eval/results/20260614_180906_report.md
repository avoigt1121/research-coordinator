## PDAC vs Chronic Pancreatitis: Analysis Not Possible

### Why this comparison cannot be run

The only registered dataset containing any chronic pancreatitis (CP) tissue is **gse71989_chen** (Chen et al., GSE71989, GPL570 Affymetrix microarray). However, it contains only **1 CP sample** alongside 13 PDAC and 8 normal pancreatic tissue samples.

Differential expression analysis (limma, ttest, or DESeq2) requires a **minimum of 3 samples per group** to estimate within-group variance. With n=1 in the CP group, no statistical test can be performed — there is no variance to model, and any "comparison" would be meaningless.

### What IS available in gse71989_chen

| Group | n |
|---|---|
| PDAC | 13 |
| Normal pancreatic tissue | 8 |
| Chronic pancreatitis | **1** ← insufficient |

### What CAN be done

### Recommendation
